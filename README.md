# Hot Dog Stand - Windows 3.1 Throwback Theme for Omarchy

## To install:
1. Copy the theme repository URL
2. Click `SUPER + ALT + SPACE`
3. Click `Install`
4. Click `Style`
5. Click `Theme`
6. Paste the link and hit enter to submit

### Alternative Install
```bash
omarchy-theme-install https://github.com/zackkitzmiller/omarchy-hotdog-stand-theme
```
